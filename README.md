A simple and beginner-friendly Calculator Application built using Java, Swing, and AWT.
This GUI-based project performs basic arithmetic operations and is perfect for learning:

Java GUI programming
Event handling
JButton actions
JTextField input/output

✨ Features
✔ Add two numbers
✔ Subtract two numbers
✔ Multiply two numbers
✔ Divide two numbers
✔ Input validation (handles errors)
✔ Clean and simple GUI
✔ Built using Java Swing & AWT

This project is created for learning Java GUI (Swing) and can be extended with:
Dark Mode
Scientific calculator functions
Better UI
